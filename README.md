# LostFoundApp

## Deskripsi
LostFoundApp adalah aplikasi Java berbasis desktop untuk mencatat dan mengelola data barang hilang menggunakan file CSV.

## Fitur
- Input data barang hilang
- Menyimpan data ke file CSV
- Menampilkan daftar barang hilang

## Struktur Project
LostFoundApp
├── BarangHilang.java
├── DataManager.java
├── DashboardFrame.java
├── FormFrame.java
├── ListFrame.java
├── HistoryFrame.java
├── MainApp.java
└── data_barang_hilang.csv

## Cara Menjalankan
1. Buka project di IntelliJ IDEA
2. Jalankan class `MainApp`
3. Aplikasi akan tampil

## Teknologi
- Java
- Java Swing
- CSV
